Recent Repositories

Repository 1

Movie Reservation System (with analytics)

The aim of this model is to create a movie reservation system which has features for two types of users: admin and the customer. The customer would be able to book tickets for a movie in a step-by-step manner (select movie, theatre, showtime). For the admin, features like being able to create slots for new movies, update price, delete movies, shows and tickets. The model would interact with a database and query on the same using a high-level language (python). At the end, a report consisting of statistics related to the theatre database for the admin using plots and table as data analytics are shown.
